# Nilesoft Shell Theming

Place `./theme.nss` in `imports` directory right next to the `shell.nss` configuration file.

When Nilesoft shell is installed with scoop package manager, this directory is present in `%USERPROFILE%\scoop\apps\nilesoft-shell\current\imports`

Simply replace the old `theme.nss` with this one.

This theme fixes the opacity and forces dark mode at all times.
